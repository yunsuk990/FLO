# 제공되는 서비스

#### 2. 맨몸 운동에 대한 자세 평가 기능
#### 3. 라이브러리를 통한 운동가이드 제공
#### 4. 즐겨찾기 등록, 삭제, 조회 기능
#### 5. 캘린더를 통한 운동계획
